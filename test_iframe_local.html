🧪 Prueba de Integración - AeropuertosGlobal

📍 URL de la aplicación: https://aeropuertos-web.onrender.com
📝 Propósito: Verificar que el iframe funciona correctamente antes de SharePoint
⏱️ Primera carga: Puede tardar 30-60 segundos si el servicio estaba dormido

✅ Checklist de Verificación

1. El iframe carga correctamente
Deberías ver el mapa con el título "SITA Mexico PNR3 H12026"
2. El mapa base de MapTiler se ve
Se ve un mapa oscuro del mundo con países y océanos
3. Los aeropuertos aparecen como círculos
Círculos naranjas/amarillos de diferentes tamaños según volumen
4. Las rutas de vuelo se muestran
Líneas azules conectando aeropuertos
5. El panel lateral funciona
Muestra estadísticas: 223,248 vuelos, 35.0M pasajeros, etc.
6. Los filtros funcionan
Puedes cambiar el filtro de rutas (Top 200, Top 100, etc.)
⚠️ NOTA: Si ves un mensaje de error o la página tarda mucho en cargar:
  • Espera 60 segundos (el servicio puede estar "despertando")
  • Recarga la página (F5)
  • Verifica que Render.com esté activo: Dashboard Render
